使い方:
1. chrome://extensions を開く
2. 「デベロッパーモード」を ON
3. 「パッケージ化されていない拡張機能を読み込む」からこのフォルダを選択
4. https://korimizu.com/CPCC/index.php を開く
5. 右下の「最適デッキ」を押す
6. 「カード再読込」→「全ワーク計算」または各ワークボタンを押す

注意:
- 現在は生徒会室とたまり場の場効果を無視する前提です。
- オプション % は「そのカードの最終Powerへそのまま乗る」式で仮実装です。
- 実ゲームと1〜2点ずれる場合は content.js 内の evaluateDeck() を調整してください。
- DOM構造が変わった場合は parseOwnedCards() 周辺の調整が必要です。
